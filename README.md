## ローカルで編集する場合

Python 3.0系インストールされている場合

```
$ sh run-py3server.sh
```

Python 2.0系インストールされている場合

```
$ sh run-py2server.sh
```

Ruby インストールされている場合

```
$ sh run-rbserver.sh
```

PHP インストールされている場合

```
$ sh run-phpserver.sh
```
